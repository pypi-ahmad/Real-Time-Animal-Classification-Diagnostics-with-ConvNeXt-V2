# Real-Time-Animal-Classification-Diagnostics-with-ConvNeXt-V2

## Overview

This repository implements an image-classification system for 10 animal classes using PyTorch and Streamlit.

### Installation

```bash
uv venv
uv pip install -r requirements.txt
```

## Usage

Run the primary app with `uv run app.py`.

## Testing
